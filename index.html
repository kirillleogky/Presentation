What is the Grid?

“This CSS module defines a two-dimensional grid-based layout system, optimized for user interface design. In the grid layout model, the children of a grid container can be positioned into arbitrary slots in a predefined flexible or fixed-size layout grid.”

“It is a 2-dimensional layout system, meaning it can handle both columns and rows, unlike flexbox which is largely a 1-dimensional system. ”

Definitions in CSS Grid Layout

Grid Lines

Grid Lines

- invisible horizontal and vertical dividing lines, they exist on both sides of a row and column.

Definitions in CSS Grid Layout

Grid Track

Grid Track

- space between two adjacent grid lines.

Definitions in CSS Grid Layout

Grid Cell

Grid Cell

- space which is bounded by four grid lines, similar to a table cell.
